# TestForm 模块使用说明

## 概述

TestForm是一个基于Vue3 + Element Plus的5步测评表单模块，专为游戏心理健康测评设计。

## 功能特性

### ✅ 已实现功能

- **5步表单流程**：基础信息 → 游戏行为 → 生活习惯 → 社交学业 → 体感生理
- **实时校验**：空值检测、数值范围限制、必填项验证
- **数据持久化**：localStorage自动保存，24小时内可续填
- **响应式设计**：完美适配桌面端和移动端
- **进度管理**：可视化进度条和步骤指示器
- **智能导航**：步骤切换验证，前置步骤完成检查

### 🎯 核心组件

#### 1. 入口组件 (`index.vue`)

- 进度条和步骤指示器
- 表单导航和验证逻辑
- 自动保存提示

#### 2. 步骤组件

- **Step1.vue**：基础信息（年龄、性别、收入、BMI）
- **Step2.vue**：游戏行为数据（17个字段，滑块/下拉/选择器）
- **Step3.vue**：生活习惯（睡眠、饮食、运动）
- **Step4.vue**：社交学业（社交关系、学业表现）
- **Step5.vue**：体感生理（可选填写，身体感觉、生理健康）

#### 3. 通用组件 (`FormItem.vue`)

- 支持多种输入类型：input、number、slider、select、radio、rating
- 内置验证规则和错误提示
- 统一的UI风格和交互体验

#### 4. 状态管理 (`testFormStore.js`)

- Pinia状态管理
- localStorage数据持久化
- 表单验证状态跟踪

## 使用方法

### 1. 路由访问

```
http://localhost:3007/test
```

### 2. 表单流程

1. **步骤1**：填写基础信息（必填）
2. **步骤2**：描述游戏行为模式（必填，17个字段）
3. **步骤3**：评估生活习惯（必填，7个字段）
4. **步骤4**：评价社交和学业情况（必填，8个字段）
5. **步骤5**：身体感觉和生理状况（可选，11个字段）

### 3. 数据保存

- 每步操作自动保存到localStorage
- 24小时内可续填未完成的内容
- 提交后清除本地数据

## 技术实现

### 表单验证

```javascript
// 实时校验示例
const rules = {
  age: [
    { required: true, message: "请输入年龄", trigger: "blur" },
    {
      type: "number",
      min: 1,
      max: 120,
      message: "年龄范围：1-120岁",
      trigger: "blur",
    },
  ],
};
```

### 数据持久化

```javascript
// 自动保存到localStorage
const saveToStorage = () => {
  const dataToSave = {
    formData: { ...formData },
    currentStep: currentStep.value,
    timestamp: Date.now(),
  };
  localStorage.setItem("gameMentalTestForm", JSON.stringify(dataToSave));
};
```

### 响应式交互

```vue
<!-- 滑块组件示例 -->
<FormItem
  label="日均游戏时长"
  type="slider"
  v-model="formData.dailyGameHours"
  :min="0"
  :max="24"
  :step="0.5"
  unit="小时"
  :required="true"
/>
```

## 字段说明

### Step1：基础信息

- `age`：年龄（1-120岁）
- `gender`：性别（男/女/其他）
- `income`：月收入范围
- `bmi`：BMI指数（10-50）

### Step2：游戏行为数据（17个字段）

- 时长相关：日均、周末、连续时长
- 频率相关：每周天数、频率类型
- 偏好相关：游戏类型、平台
- 消费相关：月均消费、冲动消费
- 社交相关：游戏内交流、语音时长
- 时间分布：主要时段、熬夜频率
- 强度评估：沉浸度、成瘾倾向

### Step3：生活习惯（7个字段）

- 睡眠：时长、质量、困难程度
- 饮食：规律程度、健康程度
- 运动：时长、频率

### Step4：社交学业（8个字段）

- 社交：活跃度、满意度、孤独感
- 学业：动力、效率、压力
- 校园：归属感、活动参与

### Step5：体感生理（11个可选字段）

- 身体感觉：疲劳、头痛、睡眠障碍、食欲变化
- 生理健康：整体健康、慢性疾病、药物使用
- 心理生理：心跳、出汗、肌肉紧张、胃肠不适
